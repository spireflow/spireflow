# Spireflow Components Library

This Storybook showcases components made with **Shadcn UI** and charts built on **Recharts** library, both customized with **Spireflow's color palette**.

All components are fully typed in **TypeScript**, with **accessibility** provided by **Radix primitives**.

## What's included

- **Theme** — Color palette for light and dark modes
- **UI Elements** — Alerts, avatars, badges, buttons, dialogs, popovers, progress, separators, skeletons, tabs, tooltips
- **Forms** — Checkboxes, inputs, radio groups, selects, sliders, switches, textareas
- **Charts** — Line, area, bar, stacked bar, pie, donut, radar, scatter, composed charts

## Theme support

All components support both **light** and **dark** themes. Use the theme switcher in the toolbar to preview components in different modes.
